# 你好，世界，你好 Memfit！

完成安装和 AI 配置后，让我们正式开始使用 Memfit。本节将带你完成第一次与 Memfit AI Agent 的交互，并认识它的核心界面和三大系统。

## 进入 AI Agent

启动 Memfit 后，进入 AI Agent 页面。这是你与 Memfit 交互的主要界面。

![image-20260316121517301](/img/help/image-20260316121517301.png)

界面由以下几个区域组成：

- **对话区域**: 中央的对话框，你可以在这里输入任务描述或问题
- **首页推荐**: 右侧展示系统推荐的工具、技能和知识库，帮助你快速上手
- **知识库 / 技能库 / 工具库**: 左下方的管理面板，用于查看和管理已加载的资源

## 发送第一条消息

在对话框中输入你的第一条消息，和 Memfit 打个招呼：

```
你好，请介绍一下你自己，你能做什么？
```

Memfit 会回应你，介绍它的能力和可用的工具。你可以看到 AI 的思考过程和最终回答。

![image-20260316121603838](/img/help/image-20260316121603838.png)

:::tip 提示
Memfit 的对话输入支持 `Shift + Enter` 换行，按 `Enter` 或点击发送按钮即可发送消息。
:::

## 认识首页推荐

在 AI Agent 主界面的右侧，你会看到 **首页推荐** 区域，它展示了三类可用资源：

### 工具 (Tools)

工具是 Memfit 可以调用的原子操作能力。每个工具完成一个具体的功能，例如：

- **SYN+TCP 指纹扫描**: 对目标进行端口扫描和服务识别
- **TLS 证书检查**: 检查目标的 TLS/SSL 证书信息
- **Cybersecurity Risk Output Tool**: 输出结构化的安全风险报告

你可以直接点击推荐的工具来触发相关任务，也可以在对话中自然描述你的需求，Memfit 会自动选择合适的工具。

![image-20260316121732130](/img/help/image-20260316121732130.png)

### 技能 (Skills)

技能是面向特定场景的能力包，集成了专门的提示词、工具组合和领域知识。例如：

- **SyntaxFlow 规则补全**: 辅助编写 SyntaxFlow 代码审计规则
- **XSS 漏洞检测**: 针对 XSS 漏洞的专项检测能力

技能让 Memfit 在特定场景下表现得像一个领域专家。

![image-20260316121744542](/img/help/image-20260316121744542.png)

### 知识库 (Knowledge Base)

知识库为 AI 提供领域专业知识的支撑。你可以挂载不同的知识库，让 Memfit 在回答和执行任务时参考这些专业资料。

![image-20260316121810590](/img/help/image-20260316121810590.png)

## 尝试一个真实任务

现在，让我们给 Memfit 一个真实的任务。你可以直接在对话框中描述：

```
帮我检查 example.com 的 TLS 证书信息
```

观察 Memfit 的执行过程 -- 它会经历 **思考、行动、观察** 的循环：

1. **思考**: AI 分析你的需求，判断应该使用哪个工具
2. **行动**: 调用 TLS 证书检查工具，发起实际请求
3. **观察**: 获取结果并分析
4. **回答**: 将发现整理成清晰的报告反馈给你

整个过程在界面上都是可见的，这就是 Memfit 的理念 -- **看得见的行动力**。

![image-20260316122018721](/img/help/image-20260316122018721.png)

## 了解三大核心系统

在日常使用中，你会频繁与 Memfit 的三大核心系统打交道。让我们逐一认识它们。

### AI Agent 执行引擎

这是 Memfit 的核心，驱动 AI 完成从简单查询到复杂多步骤任务的全部工作。

执行引擎支持两种模式：
- **即时执行**: 对于明确的单步任务，AI 直接进入思考-行动-观察循环来完成
- **规划执行**: 对于复杂任务，AI 先制定计划、分解子任务，经你确认后再逐步执行

你不需要手动选择模式，Memfit 会根据任务复杂度自动判断。

![image-20260316122228340](/img/help/image-20260316122228340.png)

### 知识库系统

知识库系统让你可以为 AI 提供专业领域的参考资料。

![image-20260316122249104](/img/help/image-20260316122249104.png)

你可以在知识库中：
- **手动添加**知识条目
- **外部导入**已有的文档和数据
- **管理和组织**不同主题的知识库
- 在与 AI 对话时**挂载**特定知识库，让 AI 在回答时参考这些资料

知识库让 Memfit 不仅仅依赖 AI 模型自身的知识，还能结合你提供的专业资料来工作。

### 记忆库系统

记忆库是 Memfit 区别于普通 AI 对话工具的关键特性之一。

![image-20260316122316802](/img/help/image-20260316122316802.png)

记忆库的特点：
- **自动记忆**: Memfit 会在交互过程中自动提炼和记忆重要信息
- **长期保持**: 记忆跨会话保留，不会因为关闭对话而丢失
- **智能召回**: 在后续任务中，Memfit 会自动关联相关记忆，提供更精准的服务
- **质量评估**: 每条记忆都有质量评分，系统自动管理记忆的优先级

简单来说，Memfit 越用越懂你 -- 它会记住你的偏好、历史操作和领域上下文，让每次交互都更高效。
